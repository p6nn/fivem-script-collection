# collection of fivem script some with previews

pen-parkingmeters - parking meter script, syncs with server

pen-roaddamage - roads get damaged upon more use, syncs with server and database, tyres pop on chance of how damaged the road is

pen-carparts - car robber, break open hood > stripped car part > all synced to the server via plate.

pen-outages - random power outages and alerts in specific areas; gta lighting limitations cause a global blackout on entry unfortunately, but idea can be expanded on.

pen-impound - https://github.com/p6nn/pen-impound
